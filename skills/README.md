# Skills

Procedural knowledge the Hermes AI Agent can load on demand. Each skill is a
directory containing a `SKILL.md` (YAML frontmatter with `name` + `description`,
followed by instructions) and any supporting templates or scripts.

## Available skills

| Skill | Description |
| ----- | ----------- |
| [`crm`](./crm/SKILL.md) | Lightweight CRM for a solo founder — track leads, contacts, deals, log interactions, and surface daily follow-ups from a simple CSV store. |
