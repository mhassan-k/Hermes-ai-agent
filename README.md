# Hermes AI Agent

A testing repository for exploring and experimenting with **Hermes AI Agent** capabilities.

## Overview

This repository is used to test and demonstrate the features of Hermes AI Agent — an intelligent assistant built by Nous Research.

## Features

- 🤖 AI-powered task automation
- 🔧 Tool integrations (GitHub, Telegram, and more)
- 🧠 Persistent memory across sessions
- 📋 Skill-based procedural knowledge
- 🔄 Scheduled cron jobs

## Getting Started

Clone this repository:

```bash
git clone https://github.com/mhassan-k/Hermes-ai-agent.git
cd Hermes-ai-agent
```
